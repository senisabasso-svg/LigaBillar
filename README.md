# Liga Salteña de Billar - Sitio Web Oficial

Sitio web oficial de la Liga Salteña de Billar de Salto, Uruguay.

## Características

- **Diseño Responsive**: Adaptado para dispositivos móviles, tablets y escritorio
- **Navegación Intuitiva**: Menú con submenús y navegación suave entre secciones
- **Información Completa**: 
  - Autoridades y Comisión Fiscal
  - Campeonatos oficiales
  - Circuito Selectivo Salteño con reglas detalladas
  - Instituciones afiliadas
- **Formulario de Contacto**: Para comunicación con la liga

## Estructura del Proyecto

```
LigaBillar/
├── index.html      # Página principal
├── styles.css      # Estilos CSS
├── script.js       # JavaScript para interactividad
└── README.md       # Este archivo
```

## Secciones Principales

### 1. Inicio
Página de bienvenida con información destacada

### 2. Institucional
- **Autoridades**: Presidente, Vice Presidente, Secretaria, Tesorero, Vocal
- **Comisión Fiscal**: Miembros de la comisión

### 3. Campeonatos
- **Campeonato Oficial LSB 2026**
  - Preparación Individual por Series
  - Preparación Individual por Equipo
  - Campeonato Salteño (Pareja de Tercera, Segunda, Primera e Individual)
- **Campeonato Oficial Federación Uruguay de Billar**
- **Circuito Selectivo Salteño** (Enfoque principal)
  - Tercera, Segunda, Primera
  - Reglamento completo con ejemplos
  - Sistema de puntos y cálculo de promedios

### 4. Instituciones
Lista de todas las instituciones afiliadas:
- Almagro A
- Almagro B
- Salto Nuevo
- Cultura A
- Cultura B
- Sociedad Recreativa Centenario
- Círculo Sportivo

### 5. Contacto
Formulario de contacto con validación

## Circuito Selectivo Salteño

El sitio incluye una sección detallada sobre el Circuito Selectivo con:

- **Formato de Serie**: 4 jugadores por serie
- **Sistema de Eliminación**: Diagrama visual del formato
- **Puntos por Posición**:
  - 1° Lugar: 8 puntos
  - 2° Lugar: 6 puntos
  - 3° Lugar: 4 puntos
  - 4° Lugar: 2 puntos
- **Cálculo de Promedio**: Ejemplos detallados con fórmulas
- **Reglas de Desarrollo**: Paso a paso del torneo

## Uso

1. Abre `index.html` en tu navegador web
2. Navega por las diferentes secciones usando el menú
3. El sitio es completamente funcional sin necesidad de servidor

## Personalización

### Colores
Los colores principales están definidos en variables CSS en `styles.css`:
- `--primary-green`: Verde principal (#1a5f3f)
- `--light-green`: Verde claro (#4a9d6e)
- `--dark-green`: Verde oscuro (#0d3d26)

### Contenido
Edita directamente el archivo `index.html` para actualizar:
- Información de autoridades
- Resultados de campeonatos
- Posiciones del circuito selectivo
- Instituciones afiliadas
